## Scope of Support

### In Scope

We accept issues and PRs for:

- Bug reports with reproducible steps
- Documentation improvements
- Feature requests (within project scope)
- Security vulnerabilities (see [SECURITY.md](./SECURITY.md))

### Out of Scope

The following are NOT supported through this repository:

- **Gawain API issues** - Contact Gawain support directly
- **Kinosuke/subscription issues** - Contact Kinosuke support
- **STORES platform issues** - Use STORES's support channels
- **Custom integration work** - This is a reference implementation only
- **Production deployment support** - Use at your own risk

## Commercial Usage

This is a **reference implementation** for demonstration purposes.

For commercial video generation:
1. Generate a preview using this plugin
2. Visit the Kinosuke upgrade URL provided after generation
3. Subscribe to a commercial plan
4. Your `install_id` will be linked to your account
